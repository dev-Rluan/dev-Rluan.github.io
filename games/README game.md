# 냥이 가챠 🎁

간단한 웹 브라우저 고양이 뽑기(가챠) 게임입니다.  
버튼을 누르면 다양한 희귀도의 고양이 중 한 마리를 랜덤으로 뽑을 수 있습니다!

## 게임 방법
1. "냥이 뽑기!" 버튼을 클릭하세요.
2. 랜덤으로 고양이(5종류) 중 한 마리가 등장합니다.
3. 각 고양이는 희귀도가 다르며, 등장 확률도 다릅니다.

### 고양이 목록 및 희귀도
- 냥이 🐱 (일반)
- 치즈냥이 🧀 (일반)
- 흑냥이 🖤 (드물게)
- 스핑크스 😼 (희귀)
- 전설냥이 👑 (아주 희귀)

## 테스트 기능
브라우저 콘솔에서 `testGacha()`를 실행하면 1000번 뽑기 시뮬레이션 결과를 확인할 수 있습니다.
